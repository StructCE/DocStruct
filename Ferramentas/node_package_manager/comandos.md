## Comandos Basicos do NPM

### Instalar um pacote

```bash
npm install 
```
### Remover um pacote

```bash
npm remove
```

### Iniciar um projeto novo

```bash
npm init
```

### Atualiza os pacotes do projeto

```bash
npm update
```

### Lista todas as dependências do projeto

```bash
npm list
```

### Instalação limpa das dependências 

```bash
npm ci
```
