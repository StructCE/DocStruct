## Comandos para instalar o NVM

1. Instalar o Curl

```bash
sudo apt update
sudo apt install curl
```

2. Instalar o NVM

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
```

3. Reinicie o terminal
